# 🖼️ Image Classification using CNN

## 📌 Overview
This project implements a Convolutional Neural Network (CNN) to classify images from the CIFAR-10 dataset into 10 categories.

## 🚀 Features
- Image classification using deep learning
- CNN architecture with Conv2D and MaxPooling
- Model training and evaluation
- Prediction on test images

## 🧠 Approach
- Data normalization
- CNN model with multiple convolution layers
- Activation function: ReLU
- Output layer: Softmax
- Optimization using Adam

## 📊 Model Performance
- Training Accuracy: ~73%
- Test Accuracy: ~71%

## 🛠️ Tech Stack
- Python
- TensorFlow / Keras
- NumPy

## 📈 Workflow
Image → CNN → Feature Extraction → Classification

## 🔮 Future Improvements
- Use Transfer Learning (ResNet, MobileNet)
- Data Augmentation
- Hyperparameter tuning

Open the notebook and run all cells.
